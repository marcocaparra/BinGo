# ♻️ BinGo  

O **BinGo** é um projeto de inovação sustentável que transforma o descarte de lixo eletrônico em uma experiência gamificada.  
Nosso objetivo é conscientizar a sociedade sobre o impacto ambiental do e-lixo, incentivando o descarte correto por meio de **missões, pontuações, rankings e recompensas**.  

---

## 🚀 Motivação  
Todos os anos, milhões de toneladas de lixo eletrônico são descartadas de forma incorreta, contaminando o solo, a água e o ar.  
O **BinGo** nasceu para mudar isso, tornando o descarte uma ação **simples, divertida e recompensadora**.  

---

## 🛠️ Tecnologias Utilizadas  
- **Software**: Python + FastAPI (back-end), banco de dados (a definir), integração com app/web  
- **Gamificação**: Sistema de pontos, missões, recompensas e ranking de usuários  

---

## ⚙️ Como Funciona  
1. O usuário descarta corretamente um dispositivo eletrônico no ponto de coleta.  
2. O sistema do BinGo identifica o descarte via hardware (Arduino + sensores).  
3. O usuário recebe pontos e desbloqueia missões no sistema.  
4. Rankings e recompensas incentivam a participação contínua.
