# ♻️ BinGo — Aqui, descartar lixo é coisa de herói!

BinGo é uma plataforma gamificada de descarte consciente de lixo eletrônico, voltada especialmente para escolas e instituições de ensino. Nosso objetivo é transformar o ato de reciclar em uma experiência divertida, educativa e recompensadora.

## 🚀 Objetivo

O BinGo nasceu com a missão de resolver um problema real: o descarte incorreto de lixo eletrônico. Segundo a *The Global E-Waste Monitor (2020)*, o Brasil é o 5º maior produtor desse tipo de resíduo no mundo, mas recicla menos de **3%**.

Nosso projeto busca resolver isso com uma abordagem moderna e motivadora, usando **gamificação** para engajar os estudantes e criar uma geração mais consciente.

---

## 🔧 Como funciona?

O BinGo combina **hardware + software** para criar um sistema interativo de reciclagem:

### 🛠️ Estrutura Física
- **Arduino Uno**
- **2 sensores IR** (detectam o descarte)
- **Display I2C LCD** (exibe mensagens e código do usuário)
- **Buzzer passivo** (feedback sonoro)

### 💻 Plataforma Web
- Tela de cadastro e login
- Geração de perfil com XP, nível e conquistas
- Ranking entre alunos/turmas
- Registro de descartes com códigos únicos

---

## 🎮 Funcionalidades

- Código único gerado a cada descarte
- Ranking e sistema de pontos (Bins)
- Recompensas físicas (entregues pela escola)
- Missões, conquistas e batalhas entre turmas
- Validação por professores e coordenadores educacionais
- Modelo de assinatura mensal para instituições
